# OpenParliamentTV-Data-DE-NW
Open Parliament TV Data Repository for the North Rhine-Westphalia state parliament (Landtag Nordrhein-Westfalen)

> **Status:** in development

## Pipeline

This data is generated by the `DE-NW` parliament implementation in
[OpenParliamentTV-Tools](https://github.com/OpenParliamentTV/OpenParliamentTV-Tools):
[`optv/parliaments/DE-NW`](https://github.com/OpenParliamentTV/OpenParliamentTV-Tools/blob/main/optv/parliaments/DE-NW).
